
# SVGWorld

A one Stop Solution to get all your SVGs. 


## Tech Stack

 
- **Frontend Framework:** React
- **Styling:**  MaterialUI
- **Backend** Nodejs
- **WebScraping** Puppeteer and Browserless.io
  




## Run Locally

Clone the project

```bash
  git@github.com:coderkhushal/SVGWorld-browserless.git
```

Go to the project directory

```bash
  cd SVGWorld-browserless
```

Install dependencies

```bash
  npm install
```

Start React application

```bash
  npm start
```
## Starting Backend

Run docker image
```bash
$ docker run \
  --rm \
  -p 3000:3000 \
  -e "CONCURRENT=10" \
  -e "TOKEN=6R0W53R135510" \
  ghcr.io/browserless/chromium
```

Go to NodeJs server directory
```bash
cd SVGWorld-browserless/server
```
Install dependencies
```bash
npm install
```
Run server
```bash
node ./index.js
```
